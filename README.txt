UPLOAD TO GITHUB
1. Unzip this package.
2. Open your GitHub public repository.
3. Click Add file > Upload files.
4. Upload index.html, styles.css, script.js and the complete assets folder.
5. Click Commit changes.

CLOUDFLARE PAGES
1. Cloudflare > Workers & Pages > Create > Pages > Connect to Git.
2. Select your repository.
3. Framework preset: None.
4. Build command: leave blank.
5. Build output directory: /
6. Deploy.

IMPORTANT: The newly requested oil-analysis product has been added as a product card with a placeholder illustration. Replace assets/oil-analysis-placeholder.svg with your actual photo, keeping the same filename, when you have the original image file available.
